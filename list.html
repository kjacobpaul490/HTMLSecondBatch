Lists in HTML

HTML provides different types of lists to organize content in a structured manner.

Example of Unordered List

Here is an example of an unordered list:

  • Item 1
  • Item 2
    • Subitem 2.1
    • Subitem 2.2
  • Item 3

Example of Ordered List

Here is an example of an ordered list:

  1. First item
  2. Second item
    1. Subitem 2.1
    2. Subitem 2.2
  3. Third item
  4. Fourth item
    1. Subitem 4.1
    2. Subitem 4.2
