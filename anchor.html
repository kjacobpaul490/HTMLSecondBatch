Anchor Tag in HTML

The anchor tag (<a>) is used to create hyperlinks in HTML. It can link to another webpage, a different section of the same page, or even trigger downloads.

Example of Anchor Tag

Here is an example of an anchor tag linking to an external website:

Visit Example.com

Example of Anchor Tag

Here is an example of an anchor tag linking to an internal page:

Visit Sub Anchor Page
